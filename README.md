# 🚀 Portfolio using HTML & CSS only

> **Clean, Simple, and Pure HTML & CSS Portfolio**

A minimalist personal portfolio website built **entirely with HTML and CSS** — no frameworks, no JavaScript. It showcases my journey as an aspiring web developer, academic background, skills, and contact information in a clean, readable layout.

🌐 **Live Site:** [https://naimxofficial.github.io/Portfolio-using-HTML-CSS-only/](https://naimxofficial.github.io/Portfolio-using-HTML-CSS-only/)

---

## ✨ Features

| Feature              | Details |
|---------------------|--------|
| 🏠 **Header**       | Simple dark header with navigation links to all sections. |
| 👤 **About Me**     | Introduction with profile image placeholder, bio, and a fun fact sidebar. |
| 📚 **Education**    | Academic background displayed in a clean, bordered table. |
| 💡 **Skills**       | Ordered list of current skills with a collapsible "My Goal" section. |
| 📁 **Projects**     | Highlights the current portfolio as a self-referential project. |
| 📬 **Contact**      | Contact details (email, phone, location) + a functional contact form. |
| 🔄 **Scrolling Banner** | Animated marquee-style text in the footer showing last updated date. |
| 📱 **Responsive**   | Basic responsiveness with clean typography and spacing. |

---

## 🎨 Design Highlights

- **Clean & Minimalist** — Focused on content readability.
- **Light Theme** — Soft background (`#f4f6f8`) with white content cards.
- **Dark Header** — Professional contrast with light text.
- **Accent Color** — Blue tones for links and buttons.
- **Typography** — Simple, legible sans-serif font stack (Arial/Helvetica).
- **Pure HTML + CSS** — No external libraries or frameworks used.
